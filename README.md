# 🎮 Jogo da Memória de Emojis

Um desafio de memória dinâmico e profissional, desenvolvido com foco em interatividade, persistência de dados e experiência do usuário. O jogo oferece uma jornada completa, desde a escolha de temas até a disputa por posições no ranking.

## 🚀 Demonstração

O projeto está hospedado e pronto para ser testado. Você pode jogar clicando [aqui](https://jucianasoares.github.io/Jogo-da-mem-ria-de-emojis/).

---

## ✨ Funcionalidades Principais

- **Dificuldade Progressiva:** O jogo se adapta conforme você avança (Fácil, Médio e Difícil).
- **Temas Customizáveis:** Interface dinâmica que permite trocar o visual e os ícones das cartas.
- **Sistema de Ranking:** Armazenamento de pontuações para monitorar o desempenho.
- **Exportação de Dados:** Funcionalidade para exportar o ranking em formato **JSON**, permitindo a portabilidade dos dados.
- **Botão Jogar Novamente:** Fluxo de jogo contínuo para reiniciar a partida rapidamente após a vitória.
- **Feedback Sonoro:** Sons para acertos, erros, alertas de tempo e vitórias (otimizado para dispositivos mobile).
- **Responsividade:** Totalmente adaptado para telas de celulares, tablets e computadores.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura e elementos de áudio/vídeo.
- **CSS3:** Animações de flip, estilização de temas e layout flexível.
- **JavaScript (ES6+):** Motor do jogo, lógica de ranking e exportação de arquivos.
- **JSON:** Formato utilizado para estruturar e exportar os dados do ranking.
- **GitHub Pages:** Hospedagem do projeto.

## 🔧 Diferenciais Técnicos

Durante o desenvolvimento, foram aplicadas soluções avançadas para garantir a qualidade:
- **Persistência de Dados:** Gerenciamento de ranking para salvar o progresso do jogador.
- **Tratamento de Áudio:** Uso de `load()` e `playPromise` para contornar restrições de reprodução automática em navegadores como Edge e Chrome.
- **Padronização de Arquivos:** Organização rigorosa de diretórios para compatibilidade com servidores Linux (case-sensitivity).
